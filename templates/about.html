AreaGrade Project

AreaGrade helps you find the best areas to rent in Antwerp based on public transport accessibility.

This project analyzes OpenStreetMap data for Antwerp to evaluate areas based on their proximity to:

  • Tram stops and lines
  • Bus stops and routes
  • Velo bike sharing stations

Each location is given a grade from A+ to F based on its overall public transport accessibility score.

About the Data

AreaGrade uses the Antwerp OSM Navigator dataset from Hugging Face to evaluate areas. The dataset contains transportation-related nodes from OpenStreetMap.

Features
  • Interactive map showing transportation options
  • Detailed scoring for bus, tram, and Velo access
  • AI-powered insights on area accessibility
  • Mobile-friendly design for on-the-go searches

This project was created as part of the Hugging Face Hackathon, focusing on using AI to solve urban transportation challenges.
